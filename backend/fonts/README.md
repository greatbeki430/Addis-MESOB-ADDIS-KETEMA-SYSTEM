# Font Files for PDF Generation

## Download Required Fonts

1. **Noto Sans Ethiopic**
   - Download from: https://fonts.google.com/noto/specimen/Noto+Sans+Ethiopic
   - Files needed:
     - NotoSansEthiopic-Regular.ttf
     - NotoSansEthiopic-Bold.ttf

2. **Roboto**
   - Download from: https://fonts.google.com/specimen/Roboto
   - Files needed:
     - Roboto-Regular.ttf
     - Roboto-Bold.ttf
